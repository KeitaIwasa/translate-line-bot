# 利用規約

本利用規約（以下「本規約」といいます。）は、岩佐デジタル（以下「当方」といいます。）が提供する「KOTORI - AI翻訳/การแปล」（以下「本サービス」といいます。）の利用条件を定めるものです。本サービスを利用するすべての利用者は、本規約に同意のうえ本サービスを利用するものとします。

## 第1条（適用）

1. 本規約は、本サービスの提供条件および本サービスの利用に関する当方と利用者との一切の関係に適用されます。
2. 当方が本サービスに関して本規約のほかに定める各種ガイドライン、ポリシー等は、本規約の一部を構成するものとします。
3. 本規約と前項のガイドライン等の内容が矛盾する場合は、特段の定めがない限り、ガイドライン等の定めが優先して適用されるものとします。

## 第2条（定義）

本規約において使用する用語の定義は、次のとおりとします。

1. 「本サービス」とは、LINE上で動作する多言語翻訳ボットおよびこれに関連するWebサイト・管理画面等一式をいいます。  
2. 「利用者」とは、本サービスを利用するすべての個人または法人をいいます。  
3. 「グループ」とは、LINE株式会社が提供するLINEサービス上のグループトークルームをいいます。  
4. 「Freeプラン」とは、当方が無償で提供するプランをいいます。  
5. 「Proプラン」とは、当方が有償で提供するサブスクリプションプランをいいます。  
6. 「決済事業者」とは、Stripe社その他当方が指定する決済サービス提供者をいいます。  
7. 「代表者」とは、本サービスをLINEグループに導入し、当該グループについてProプランその他有償サービスの申込み・契約・解約等の手続を行う利用者をいいます。

## 第3条（本サービスの内容）

1. 本サービスは、LINEグループ内で送信されたメッセージを、利用者が設定した言語に自動翻訳して表示する機能を提供します。
2. 本サービスは、翻訳精度向上および文脈理解のため、グループ内の会話履歴の一部および関連情報をサーバ上に保存し、外部のAIモデル等に送信することがあります。詳細はプライバシーポリシーに定めるとおりとします。
3. 当方は、FreeプランおよびProプランの内容、提供機能、制限事項を任意に定め、これを変更することができます。最新の内容は本サービスのWebサイト上に表示するものとします。

## 第4条（利用条件）

1. 利用者は、本規約およびプライバシーポリシーに同意したうえで本サービスを利用するものとします。
2. 利用者が未成年者である場合には、法定代理人の同意を得た上で本サービスを利用するものとします。
3. 利用者は、本サービスを利用するために必要な通信機器、ソフトウェア、通信回線等を自己の費用と責任において準備・維持するものとします。

## 第5条（アカウント・グループ情報の管理）

1. 当方は、LINEユーザーID、グループID等をキーとして、本サービス上の利用者およびグループを識別します。  
2. 利用者は、自己のLINEアカウントの管理について一切の責任を負うものとし、第三者による不正利用について当方は責任を負いません。  
3. 利用者は、所属するグループ内で本サービスを利用するについて、当該グループのメンバーその他関係者の承諾が必要な場合には、自己の責任と費用においてこれを得るものとします。  
4. 代表者は、本サービスをグループに導入するにあたり、当該グループ内のメンバー全員が、本サービスによるメッセージ内容の取得・保存・翻訳処理および外部AIモデルへの送信等、本サービスの利用に伴う情報の取扱いについて、事前に説明を受け、同意していることを保証するものとします。

## 第6条（料金および支払方法）

1. Freeプランは、当方が別途定める範囲内で無償にて提供されます。
2. Proプランは、1グループあたり月額380円（税込）のサブスクリプション料金にて提供されます。料金額は当方が任意に変更することができ、変更後の料金は当方所定の方法により利用者へ通知した時点または当方が定める日から適用されるものとします。
3. Proプランの料金は、決済事業者が提供する支払手段（クレジットカード等）を通じて支払われるものとします。
4. Proプランの利用料金は、初回申込時に1か月分が課金され、その後は毎月自動的に更新・課金されます。
5. 利用者と決済事業者との間で紛争が生じた場合、当該紛争は利用者と決済事業者との間で解決するものとし、当方は一切責任を負いません。

1. Proプランの契約期間は、申込日を起算日とする1か月間とし、その後、利用者が解約手続きを行わない限り、同条件で自動更新されるものとします。

1. 当方は、本サービスの提供のため、利用者が本サービスを通じて送受信するメッセージ内容、翻訳結果および関連情報をデータベース（Neon等）に保存します。

3. 会話データの保存期間、利用目的、第三者提供等の詳細については、別途定めるプライバシーポリシーに従うものとします。  
4. 代表者は、本サービスをグループに導入し利用させるにあたり、前各項に定める会話データの取得・保存および外部AIモデルへの送信について、当該グループのメンバー全員から事前に同意を得ていることを保証するものとします。  
5. 利用者は、本サービスを利用することにより、上記の会話データの保存およびAIモデルへの送信に同意したものとみなされます。
