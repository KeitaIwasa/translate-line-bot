# Proプラン申し込みページ仕様
Proプランに申し込むときに、Stripeの決済ページの前に表示する案内ページの仕様
- チェックアウトのIDをパラメータに含めたURLを「Proプランに申し込む」ボタンに設定する。パラメータが存在しない場合はStripe誘導ボタンを表示しない。
- 日本語、中国語（台湾）、英語、タイ語に対応する。

**見出し**

> Proプランでできること

**本文**

* グループ内のメッセージを、登録された言語（最大5言語）へ即時翻訳

* LLMが対応する幅広い言語を利用可能
* 多国籍ビジネスチームから友人グループまで、場面を選ばず利用可能

## セクション3：料金・課金条件

**見出し**

> 料金・課金条件

**本文**

* プラン名：LINE多言語翻訳AIボット Proプラン
* 料金：**月額 ¥380（税込） / 1グループ**
* 課金方式：代表者1人がクレジットカード決済（自動更新）
* 請求タイミング：初回ご購入時に1か月分をお支払い、その後は毎月自動更新されます。
* 解約方法：

  * グループ内でボットにメンションし、所定のコマンドを送信することで、いつでも解約手続きが可能です。
  * 解約手続き後も、支払済み期間の満了日までは Pro プランの機能を利用できます。
* グループからボットが退会させられた場合：

  * 当該グループの Pro プランは自動的に解約されます。
  * 期間途中でボットが退会させられた場合でも、すでにお支払いいただいた料金の返金は行いません。

## セクション4：利用開始までの流れ

**見出し**

> ご利用開始までの流れ

**本文**

1. LINEで翻訳AIボットを対象グループに招待します。
2. Freeプランとして翻訳の挙動をお試しいただけます。
3. 本ページの「Proプランを購入」ボタンから決済ページへ進み、クレジットカードでお支払いください。
4. 決済完了後、数分以内にグループの Pro プランが有効化され、月額期間中は Pro プランの機能をご利用いただけます。
5. 解約したい場合は、グループでボットにメンションし、解約コマンドを送信してください。支払済み期間の終了までは Pro プランの利用を継続できます。

## セクション5：注意事項・利用規約

**見出し**

> 注意事項・利用規約について

**本文**

* Proプランは、**1つのLINEグループごと**に契約されます。複数のグループでご利用になる場合は、それぞれのグループごとに契約が必要です。
* グループからボットが退会させられた場合、その時点で当該グループの Pro プランは自動的に解約されますが、**期間途中であっても料金の返金は行いません。**
* お客様による解約手続きの場合も、支払済み期間中の料金の返金は行いません。支払済み期間の満了までは Pro プランをご利用いただけます。
* サービスの仕様・料金・対応言語は、事前の予告なく変更される場合があります。
* 通信環境や外部サービス（LINE、LLM、Stripe 等）の障害により、一時的に翻訳結果の提供ができない場合があります。
* 本サービスは、翻訳品質の向上および文脈理解のため、グループ内のメッセージをデータベースに保存し、翻訳処理時には直近の会話履歴の一部を外部のAIモデル（Gemini）に送信します。
